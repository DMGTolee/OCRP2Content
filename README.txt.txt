INSTALLATION INSTRUCTIONS
-------------------------
1. Open up the "ocrp2content" folder.
2. Drag into your garrysmod/garrysmod folder.
3. Launch Garry's Mod.
4. Join the server and enjoy!

FYI
-------------------------
You must have Garry's Mod closed when dragging the files over.
If you want, you can create a addon format.
If you still see errors, post a error report through GitHub and i'll check it up.
